Calculus Rules
Derivative Rules:
  • • Power: d/dx(x^n) = nx^(n-1)
  • • Sum: d/dx(f+g) = f' + g'
  • • Product: d/dx(fg) = f'g + fg'
  • • Chain: d/dx(f(g(x))) = f'(g(x))·g'(x)
Integration Rules:
  • • Power: ∫x^n dx = x^(n+1)/(n+1)
  • • Sum: ∫(f+g) dx = ∫f dx + ∫g dx
  • • Constant: ∫cf dx = c∫f dx
  • • By parts: ∫u dv = uv - ∫v du
